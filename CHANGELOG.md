## [0.1.0] - 2026-03-26

### Added

- FastAPI app with Web UI, CLI, and API for turning source material into podcast episodes
- OpenAI-based script writing with multimodal document analysis for `pdf`, `docx`, and `pptx`
- OpenAI and ElevenLabs voice generation
- Live script and audio preview support in the UI
- URL, YouTube, pasted text, and file-upload ingestion paths
